# ♡ Smart Personal Study Assistant ♡

Welcome to the technical heart of my AI Lab project! This assistant was designed to bridge the gap between "information overload" and "aesthetic learning," specifically tailored for Girls in STEM who want their tools to be as cute as they are smart.

## ✨ Unique Features
- **Smart Triage:** Instantly converts massive textbook walls into 3–7 core key takeaways.
- **Active Recall Quizzes:** Generates practice questions on the fly to help you prep for AI and DSA exams.
- **Academic Tailoring:** Adjusts its vocabulary and depth based on whether you're a Year 1 student or a Graduate candidate.
- **Pastel Typewriter Aesthetic:** A custom-styled Streamlit interface featuring typewriter fonts and a light=pink coquette palette.
## 🛠️ Built With
- **Python:** The core logic for data processing and automation.
- **Streamlit:** Used to craft the web-based GUI with custom CSS injections.
- **Google GenAI SDK:** Powered by Gemini 2.0 Flash model (2026 standard) for rapid-fire intelligence.
- **JSON & OR-Tools:** For sturctured data handling and complex constraint solving.
